# Sarathi AI – Workforce Intelligence Assistant

A production‑quality MVP that provides context‑aware workforce intelligence for industrial settings.

## Features
- RAG‑based query answering over SOPs, manuals, and expert notes.
- Single‑screen dark‑theme UI built with Streamlit.
- Skill exposure tracking per user.
- Learning suggestions after each answer.

## Quick Start
```bash
# Navigate to project folder
cd SarathiAI
# Install dependencies
pip install -r requirements.txt
# Set OpenAI key (or compatible provider)
set OPENAI_API_KEY=YOUR_KEY   # Windows CMD
# Run the app
streamlit run frontend/app.py
```
